# Background

2048 was a popular game back in 2018, and right now I'm learning the Vue.js thus, I will make a 2048 game for fun. project address: https://github.com/zhoudl0605/2048

# Understand the project

according to the UI, we can separate the project into 3 parts:

  • Record
    • score of current game
    • best score in the history
  • Action
    • start new game
  • Game
    • gaming logic area
